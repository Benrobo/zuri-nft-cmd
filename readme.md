## Getting started

make sure you have `nodejs` installed on your pc.

1. ### Clone the repo or downlod it.
```sh
git clone https://github.com/benrobo/zuri-nft-cmd
```

2. ### Install all dependencies.
```sh
npm install
```

3. ### Start the program.

Simply run the `main.js` file.

```js
node main.js
```
